# 📘 Lista de Exercícios — JavaScript

Atividade da disciplina de **Programação para Internet (PPI)** — IFRN Campus Pau dos Ferros.

## 🎯 Objetivo

Praticar fundamentos de JavaScript: variáveis, operações, condicionais, laços e funções.

## 📂 Estrutura

Cada exercício está em um arquivo separado:

```
lista-js/
├── questao01.js
├── questao02.js
...
├── questao20.js
```

## ▶️ Como executar

No terminal, dentro da pasta:

```bash
node questao01.js
```

## 🧠 Conteúdo

* Variáveis e operações
* Entrada de dados (`readline`)
* Condicionais (`if/else`)
* Laços (`for`, `while`)
* Funções
